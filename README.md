# Kaifeng Chen — GitHub Pages portfolio

A static, responsive personal site based on the accompanying resume.

## Publish with GitHub Pages

1. Create a GitHub repository named `your-username.github.io` (or use any repository and publish from its `main` branch).
2. Upload these files to the repository root and push to GitHub.
3. In **Settings → Pages**, select **Deploy from a branch**, then choose `main` and `/ (root)`.

The site needs no build step.
